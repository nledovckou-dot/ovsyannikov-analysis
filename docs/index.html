4 неотложных действия

1. Запустить Telegram-канал + Instagram
Критический разрыв: 0 подписчиков. Все успешные бренды строят комьюнити (VOIS 288K TG, The Act 80K). Контент: backstage производства, коллаборации, новинки. Цель: 10K TG за 6 месяцев. Цена входа: 0₽.
2. Расширить ассортимент до 50+ SKU
27 SKU vs 290 у Giardino Magico. Добавить: гель для душа в подарочных форматах, свечи (маржа 70%+), мини-наборы для отелей. Кросс-селл повышает средний чек B2B на 30-40%.
3. Защититься от The Act в HoReCa
The Act уже продаёт диспенсеры для отелей с Gastreet и METRO. Их козырь: «бренд, который ваши гости уже знают». Ответ: эксклюзивность + кастомизация + премиальное качество, которое масс-бренд не может повторить.
4. Выйти в СНГ (Казахстан)
Экспорт российской косметики в Казахстан +19% в 2025. Спрос удвоился. Минпромторг планирует транспортные субсидии для экспортёров косметики. Окно возможностей 2026-2027.

3 стратегических хода

1

Контент-машина

VOIS доказал: 200 блогеров = 2.5 млрд выручки. Овсянникову достаточно 10 креаторов для TG + Instagram + Shorts. Backstage производства, коллаборации с отелями, рецепты ароматов.

2

Экосистема HoReCa

Диспенсеры + мыло + крем + диффузор + свечи = полная ольфакторная идентичность отеля. Tonka делает это за 8000₽/свеча. Овсянников может за 2000₽ — в 4× доступнее.

3

Дорожная карта → 500М

142М → 300М (B2B масштабирование + новые SKU) → 500М (B2C маркетплейсы + СНГ экспорт). Горизонт: 18-24 месяца. Паттерн The Act: B2C бренд → B2B продажи.

AI-анализ: почему одни умирают, а другие взлетают

Паттерны смерти

  • ✘ Зависимость от импортных поставок (Lush не смог наладить альтернативу)
  • ✘ Смерть/арест основателя (Natura Siberica, Letique)
  • ✘ Долговая нагрузка (Organic Guru — 60М+ долгов)
  • ✘ Отсутствие онлайн-каналов (The Body Shop глобально)
  • ✘ MLM-модель vs маркетплейсы (Faberlic под давлением)

Паттерны успеха

  • ✔ Маркетплейсы + контент (VOIS, Золотое Яблоко)
  • ✔ Своё производство (Mixit 1.4 млрд завод, The Act Ростов)
  • ✔ Data-driven разработка (VOIS: «голос покупателя»)
  • ✔ Тайминг импортозамещения (все ракеты стартовали 2020-2022)
  • ✔ B2B как якорь стабильности (Tonka 70+ отелей)

Вывод для Овсянникова

Овсянников следует 4 из 5 паттернов успеха (своё производство, тайминг, B2B якорь, маркетплейсы). Критический пробел — отсутствие контент-машины и digital-присутствия. Это единственный «паттерн смерти», который применим к компании.
